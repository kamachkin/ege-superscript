# Скрипт автоматической установки ПО для Windows

PowerShell скрипт для автоматизации развертывания программного обеспечения на компьютерах в школьных классах и центрах подготовки к экзаменам по информатике.

## 🚀 Возможности

- **Автоматическое подключение к сети** — подключение к Wi-Fi при отсутствии проводного соединения
- **Тихая установка ПО** — устанавливает 20+ приложений без участия пользователя
- **Умное определение** — пропускает уже установленные программы
- **Обновления Windows** — множественные циклы обновлений через PSWindowsUpdate
- **Ярлыки на рабочем столе** — автоматическое создание ярлыков для всех программ
- **Подробное логирование** — детальный лог-файл с временными метками
- **Активация Windows и Office** — опциональная активация через get.activated.win

## 📁 Подготовка

1. Создайте папку `soft` на любом диске
2. Поместите все установочные файлы в папку (см. список файлов ниже)
3. Запустите скрипт от имени администратора

## 🚦 Использование

```powershell
# Запустите от имени администратора
Set-ExecutionPolicy Bypass -Scope Process -Force
.\superscript_modified.ps1
```

Скрипт выполнит:
1. ✅ Подключение к сети (Wi-Fi при необходимости)
2. ✅ Установку NuGet провайдера
3. ✅ Установку всего ПО
4. ✅ Обновление Windows (5 циклов)
5. ✅ Установку расширений VS Code
6. ✅ Создание ярлыков на рабочем столе
7. ✅ Активацию Windows и Office
